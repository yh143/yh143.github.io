如何提交？

  1. 点击上方按钮，会打开 GitHub 的 Issue 创建页面（可能需要登录）。
  2. 页面已预填了投稿格式，请在相应位置填写视频链接和描述。
  3. 填写完毕后，点击页面底部的 「Submit new issue」 按钮即可。

此方式安全可靠，你的投稿将直接存储于本仓库的 Issues 列表 中。
